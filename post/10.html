# 前言

​ SDK，VS 等相关包傻瓜式安装，此处不做赘述，在此之前将 bin 文件添加进环境变量。

# 报错

# 一、环境配置

# 1.1 重定目标解决方案

​ 因为例程是 2015 年的，所以此处需要重定目标解决方案。此处更新到最新

​ 项目→重定目标解决方案

SDK 可以选择所下载的，平台工具及默认 V140（既 VS2015 的编译器，此处没有的话需要去下载，也可以选择最新的，但我没有测试）平台工作集也可选择

# 1.2 路径配置

项目文件右键→属性→C/C++→附加包含目录

此处包含进 SDK 中的 include 所在路径

​ 项目文件右键→属性→C/C++→附加包含目录→宏

此处的宏 realsense 默认配置路径为 program 而不是 program（x86），因此需要更改宏。

​ 视图→其他窗口→属性管理器→release→intel.realsense

更改为 SDK 所在目录。

​ 项目文件右键→属性→链接器→常规→附加库目录

添加 lib/x64 路径

​ 项目文件右键→属性→链接器→输入→附加依赖项

添加 realsense2.lib 依赖项（图中有误）

如果出现包含目录不存在等需要在各种包含中添加试验具体有以下：

属性→VC++ 目录

属性→C/C++

属性→链接器箭头→常规 / 输入→附加库目录 / 附加依赖项

属性→生成事件→后期生成事件

是否在生成时使用：否
